### Method: POST
>```
>http://localhost:3000/api/users/create
>```
### Body (**raw**)

```json
{
    "name": "postman-user",
    "email": "postman-user@test.com",
    "password": "test",
    "role": "contractor"
}
```

```json
[
    {
        "id": "15065099",
        "name": "postman-user",
        "email": "postman-user@test.com",

        "role": "contractor",
        "newUser": true,
        "emailVerified": null,
        "createdAt": "2024-06-07T02:36:38.593Z",
        "updatedAt": "2024-06-07T02:36:38.593Z"
    }
]
```

### Response: 501
```json
{
    "error": "User already registered"
}
```
